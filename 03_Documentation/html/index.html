IGM - ReConBot Documentation

The ReConBot Package includes the dynamixel_controllers package for the Dynamixel Motors (https://github.com/arebgun/dynamixel_motor) and the reconbot_control package developed at IGM (RWTH Aachen University), this package provides a main class called ReConBot.h that allows other program to establish communication with the MX-64AT motors as well as planning trajectories for each of the arms of the ReConBot or the whole group. In addition, this package allows the user to establish communication with other nodes, running in MATLAB for instance, throught specific topics.
